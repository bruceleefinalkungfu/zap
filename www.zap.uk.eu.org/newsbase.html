Zap mailing lists and Newsbase

If you want Newsbase to filter the Zap mailing lists automatically for you into different inboxes, you'll need to do the following. First, create or load !Newsbase.Support.startup.c99nonstd, and ensure that it contains: 

add header mailing-list
add header delivered-to

Now run Newsbase. When it's finished its initialisation, click Adjust on its icon and again on the News/mail filters icon.

Click on Add then the Mail option, and enter details as follows (you'll want to change the bits marked as keyboard input to match the mailing list that you are subscribing to:

  • Type = mail; continue = off
  • Name: Zap devel
  • When...
    • mailing list = * devel-help@zap.uk.eu.org; *
    • off
    • off
  • Then...
    • redirect Folder.maillist.Zap.devel
  • Filter expiry = never

Finally, click on Save.
