Download area

The current stable release is v1.35; the current beta release is v1.39. Zap distribution sets are made as 'zipchives' - Zip-format archives, compatible with the free InfoZip program (which is available for most operating systems, including Acorn RiscOS) and David Pilling's SparkFS. If you can't get either of these, download sparkplug.basic.bin, set its type to BASIC and run it - it self-extracts into a copy of SparkPlug into the current directory, which will also read zipchives.

If you need to report a bug, or want to suggest a new feature, please check the contacts page for the appropriate email address. You should also try to make sure that the bug/feature hasn't already been reported/proposed (although this is tricky at the moment because we haven't collated these).
