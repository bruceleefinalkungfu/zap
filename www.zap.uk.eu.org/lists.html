Zap mailing lists

There are a range of mailing lists pertaining to Zap, which are listed below. These lists are run using ezmlm, which is a mailing list manager you might not have come across before. Briefly, to subscribe to a list called foop@zap.uk.eu.org, send email to foop-subscribe@zap.uk.eu.org. If you want to subscribe under a different email address, use foop-subscribe-my.address=mydomain.com@zap.uk.eu.org - you need to replace the 'at' (@) in your email address with an equals sign (=). After sending your subscription request, you will be emailed to confirm the request.

ezmlm is fairly easy to use, and allows you to easily subscribe yourself under a different email address even if it is difficult for you to send from it. For more information, look at http://www.ezmlm.org/ezman/ezman1.html.

All Zap lists are 'open' except announce; that is, you don't have to be subscribed to the lists to post to them (this is particularly important for things like the bug list). announce is moderated; each message sent to it is checked first by one of the Zap development team. Please try to ensure that you send your message to the correct list (especially in the case of bugs, because they should get fixed faster if they get sent to the right place :-).

buglist
This is the list where bug reports are discussed. Please do not send bug reports to this address; use bugs@zap.uk.eu.org instead.
Subscribe to buglist
beta
This is the discussion list for people participating in beta testing of Zap. (You can 'become' a beta tester simply by downloading the latest beta version and starting to use it.) Please do not send beta bug reports to this address; use betabugs@zap.uk.eu.org instead.
Subscribe to beta
features
A largely non-technical list for discussion of proposed new features.
Subscribe to features
devel
A largely technical list for discussion of alterations and additions
to the core of Zap. Subscribe to devel
technical
Other technical issues: this is where to ask questions such as "How do I write a mode/command to do ..."
Subscribe to technical
announce (moderated)
A list to which announcements of important changes is made. In general these announcements will also be posted to comp.sys.acorn.announce.
Subscribe to announce

In addition, all the above lists except announce are available as digest lists; digests get sent every thirty messages, 64K of message bodies, or 48 hours. (A digest will actually be sent when a message arrives on the list, so low volume lists won't work well as digests; but then low volume lists don't need to work well as digests :-). To subscribe to a list as a digest, use foop-digest-subscribe@zap.uk.eu.org instead of foop-subscribe@zap.uk.eu.org

If you need to filter mail from these lists, the best way to do it is to filter on the Mailing-List or Delivered-To headers. These will appear something like below. (If you need it, we have information specific to filtering the Zap mailing lists with Newsbase.) 

Mailing-List: contact announce-help@zap.uk.eu.org; run by ezmlm
Delivered-To: mailing list announce@zap.uk.eu.org
