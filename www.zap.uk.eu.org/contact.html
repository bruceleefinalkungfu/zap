Zap contact details

We prefer to be contacted via email; if you really need to write to us, write to James at the address given below.

General email addresses. Note that the following are actually lists; for instance, the features address is a list for discussing the proposed features as well as suggesting them. For information on how to subscribe to the lists, as well as more detailed descriptions of all the Zap-related lists, see the Zap mailing lists page. Before reporting a bug or suggesting a feature, try to make sure it hasn't already been reported/proposed (this is difficult until we collate the lists of each).

bugs@zap.uk.eu.org
All bugs reports for stable releases of Zap should be sent here
betabugs@zap.uk.eu.org
All bugs reports for beta releases of Zap should be sent here
features@zap.uk.eu.org
For suggesting new features

In addition, the usual 'master' addresses exist for the Zap domain: webmaster@zap.uk.eu.org, hostmaster@zap.uk.eu.org and postmaster@zap.uk.eu.org. In addition, listmaster@zap.uk.eu.org runs the mailing lists.
