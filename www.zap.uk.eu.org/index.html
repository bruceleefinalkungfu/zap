Zap - a programmers' editor

The latest release version of Zap is 1.40. More options, and Zap extensions, are available from the download page.

  • Stable core distribution

Zap is a programmer's editor for Acorn RiscOS. It has a large number of features designed to facilitate programming (particularly source code), as well as full text editing facilities. It is also highly configurable and extensible. There is a list of frequently asked questions, and their answers.

If you wish to get in touch with the Zap development team then please use the appropriate email address from the contact page. If you wish to get involved in the development effort, you will probably want to subscribe to one or more of the mailing lists. Before reporting a bug or making a suggestion, we advise checking the Changes file in the current Zap distribution to see if it has already been dealt with. Hopefully in the future this website will contain lists of proposed features, known bugs, and recent changes - for now you'll have to download Zap itself.

We hope you find Zap a useful program.
