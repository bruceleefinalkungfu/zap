Download area

The current stable release is v1.40. An older stable release which might still be useful for some people is v1.35. The current beta release is v1.42. The current alpha version is v1.43; snapshots of its development are available.

Zap distribution sets are made as 'zipchives' - Zip-format archives, compatible with the free InfoZip program (which is available for most operating systems, including Acorn RiscOS) and David Pilling's SparkFS. If you don't have either of these, download sparkplug.basic.bin, set its type to BASIC and run it - it self-extracts into a copy of SparkPlug into the current directory, which will also read zipchives.

Please use the ftp site if you can; ftp transfer is in general faster than http. In addition, the http-based downloads may not be up to date.

If you need to report a bug, or want to suggest a new feature, please check the contacts page for the appropriate email address. You should also try to make sure that the bug/feature hasn't already been reported/proposed (the buglist is currently available via the CVS repository.

The official Zap distribution site is ftp://ftp.zap.uk.eu.org/; however this can be over-burdened at times, so we have a series of mirrors carrying the latest stable distribution. Note that these sites may not be bang up to date all the time, but will generally carry recent copies of Zap.

v1.44

Version 1.44 of Zap is now being developed. Beta releases by individual developers are available from the development release sites listed above. Slightly more stable releases will be made into the v1.44 working directory. In addition, there is a snapshots directory, which contains daily binary and source snapshots (these aren't guaranteed to work at all; if you want something that most likely will work, grab it from the main working directory).

v1.42

Version 1.42 of Zap is a public beta release; use this to try out the new features but not risk the latest development versions. It should be fairly stable, but there are no guarantees.

v1.40

Where the extension or distribution set is being maintained by one of the Zap developers, their initials are given in brackets, with a link to contact details. Otherwise, the name of the extension's author is given, with a link to contact details (if known). There is a list of frequently asked questions, and their answers.

Version 1.40 is currently at patch 9; this contains several minor bugfixes and performance improvements. A list of changes for the patches is available.
